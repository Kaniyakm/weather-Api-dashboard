# Weather Dashboard

#### A Dashboard with a Weather API integration for Global Weather

![weatherstack-forecast](https://github.com/user-attachments/assets/451f41fa-f028-4870-bbc7-d43797323bba)

## Description

A global weather dashboard with an Open-Meteo API integration that shows temperature, condition and the humidity through the Open-Meteo API

## <a name="technologiesused"></a>Technologies Used

- JavaScript
- HTML5
- CSS3
- Open-Meteo API for weather data
- Nominatim API for streetmap data

## Features

Users can type in a city name, for example "London" into the search box. The dashboard will then grab the latitude and longitude of the city from Nominatim and use that to get the weather from Open-Meteo.
